# QR Code Generator Server

A simple API server for generating QR codes.

## 💪 Features

- ✅ Asynchronous QR code generation
- ✅ Fast and lightweight
- ✅ Different formats (PNG, SVG, etc.)
- ✅ Customizable QR code size, color, etc
